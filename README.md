# python-proxy-server
CSC 430 - Caching Proxy Server
How to run:
1-Ensure to install the necessary dependencies:
   pip install flask
In addition, to access the admin interface, just open:
http://127.0.0.1:5000
2-Run proxy server
    python proxy.py
3-Configure browser proxy:
    IP: 127.0.0.1
    Port: 8888
